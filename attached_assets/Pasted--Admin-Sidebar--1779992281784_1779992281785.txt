أريد إعادة تنظيم:

# القائمة الجانبية في الأدمن (Admin Sidebar)

لأنها أصبحت طويلة جدًا ومشتتة وغير مريحة.

---

# المطلوب:

تحويل القائمة إلى:

# Professional Collapsible Sidebar

مثل:

* Stripe Dashboard
* Notion
* Vercel
* Property Finder Admin
* Shopify Admin

---

# المطلوب:

تجميع العناصر المتشابهة داخل:

# Groups / Accordion Tabs

بحيث:
عند الضغط على القسم:

* يفتح العناصر التابعة له
* وإذا أغلق يختفي المحتوى

لتقليل طول القائمة وتحسين الـ UX.

---

# مثال للتنظيم:

## 1- إدارة المستخدمين

يحتوي:

* المستخدمين
* الشركات العقارية
* الموظفين والصلاحيات

---

## 2- إدارة العقارات

يحتوي:

* العقارات
* أنواع العقار
* مميزات وخدمات العقار
* التصنيفات العقارية

---

## 3- إدارة المناطق

يحتوي:

* المحافظات والمدن والمناطق
* أهم المناطق

---

## 4- الاشتراكات والباقات

يحتوي:

* الاشتراكات
* الباقات
* الترقيات
* Boost Ads
* Featured Ads

---

## 5- الإعلانات والتسويق

يحتوي:

* إدارة الإعلانات
* البوب أب والإعلانات
* البنرات

---

## 6- التواصل والدعم

يحتوي:

* الرسائل والطلبات
* تذاكر الدعم
* الإشعارات
* البريد الإلكتروني

---

## 7- التحليلات والتقارير

يحتوي:

* التقارير
* Analytics
* تحليلات الذكاء
* الإحصائيات

---

## 8- الإعدادات

يحتوي:

* إعدادات الموقع
* إعدادات الدفع
* إعدادات الإيميلات
* SEO
* API Settings

---

# المطلوب:

* Accordion Sidebar
* Expand / Collapse
* Active State واضح
* أيقونات احترافية
* Smooth UX
* بدون فوضى

---

# التصميم:

* Admin عالمي
* Minimal جدًا
* أبيض وكحلي
* احترافي
* مرتب
* بدون زحمة

---

# مهم جدًا:

* حافظ على جميع الـ Routes الحالية
* لا تكسر أي صلاحيات
* لا تكسر الـ Permissions
* لا تكسر الـ Navigation Logic

---

# المطلوب تقنيًا:

* Dynamic Sidebar Groups
* Role Based Navigation
* Collapsible Menu System
* Reusable Sidebar Components
* Clean Navigation Architecture

---

# النتيجة:

أريد Sidebar احترافية ومنظمة جدًا بدل القائمة الطويلة الحالية، بحيث تكون سهلة وسريعة وواضحة مثل أفضل لوحات التحكم العالمية.
