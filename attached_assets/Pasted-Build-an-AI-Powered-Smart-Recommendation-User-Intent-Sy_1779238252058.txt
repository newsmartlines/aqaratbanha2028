* Search keywords
* Clicked properties
* Favorite properties
* Viewed categories
* Viewed locations
* Price ranges
* Property types

* Recent searches

Use this data to understand:

* What the user wants
* Budget level
* Preferred areas
* Preferred property type
* Buy vs Rent intent

2. Smart Recommendations Engine
   Automatically show:

* Recommended Properties
* Similar Listings
* Trending in User Area

* Recently Viewed
* Personalized Suggestions

* Hero recommendations

* Tags
