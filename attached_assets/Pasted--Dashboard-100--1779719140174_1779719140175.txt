# Final Deep Unified Dashboard Refactor

# المطلوب:

راجع الـ Dashboard بالكامل مرة أخيرة وكأنك تبني:

# Single Source Dashboard Architecture

---

# أريد التأكد أن:

* لا يوجد أي ProviderLayout منفصل
* لا يوجد أي UserLayout منفصل
* لا يوجد أي صفحات Dashboard مكررة
* لا يوجد أي Render Trees منفصلة بالكامل
* لا يوجد أي:

  * if(user.role==="user") return (...)
  * if(user.role==="provider") return (...)

تعيد بناء الصفحة بالكامل بشكل منفصل.

---

# المطلوب الصحيح:

استخدم:

* Shared Layout
* Shared Page
* Shared Component Tree

ثم داخل الصفحة نفسها:

* اعرض Sections مختلفة حسب الصلاحيات فقط.

---

# مثال:

لا أريد:

```tsx
if (user.role === "user") return <UserDashboard />
if (user.role === "provider") return <ProviderDashboard />
```

---

# أريد:

```tsx
return (
  <DashboardLayout>
    <SharedHeader />

    {isProvider && <ProviderStats />}
    {!isProvider && <UserStats />}

    <SharedContent />
  </DashboardLayout>
)
```
